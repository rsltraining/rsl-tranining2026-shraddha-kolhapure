1. Read in detail about UIView
UIView is the basic building block of the iOS user interface in UIKit. Everything visible on the screen is directly or indirectly created using UIView. It is used to display content, handle user interactions, and manage layouts on the screen.
UIView works like a container that can hold other UI elements such as labels, buttons, images, and text fields. It helps developers organize the screen properly by creating a view hierarchy. A parent view can contain multiple child views inside it.
- Showing content on the screen
- Handling touch actions
- Taking care of child views
- Supporting animations
- Controlling what is visible and how things look
UIView can be created in two ways:
1.Programmatically using code
2. Using Storyboard with drag and drop


Example:
UIView *myView = [[UIView alloc] initWithFrame:CGRectMake(50, 100, 200, 100)];
myView.backgroundColor = [UIColor redColor];
[self.view addSubview:myView];


This code creates a red UIView and displays it on the screen.


2. Read in detail about UIButton
UIButton is a user interface component in UIKit used to perform actions when the user taps on it. It is a subclass of UIView, which means it also supports properties like frame, background color, hidden, and alpha.
Buttons are commonly used in iOS applications for actions such as login, submit, register, next, save, and cancel. UIButton improves user interaction because it allows communication between the user and the application.


UIButton works using the Target-Action mechanism. A method is connected to the button, and when the button is tapped, that method is executed automatically.
Example


UIButton *button = [UIButton buttonWithType:UIButtonTypeSystem];
[button setTitle:@"Click" forState:UIControlStateNormal];
[button addTarget:self
           action:@selector(buttonClick)
 forControlEvents:UIControlEventTouchUpInside];
- (void)buttonClick {
    NSLog(@"Button Clicked");   }
This code creates a button and calls the buttonClick method when the user taps on the button.
