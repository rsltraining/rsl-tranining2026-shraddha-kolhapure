2.Read about Range operators in Swift. Give a couple of examples of usage.


Range operators in swift are used to express series of values between two values
Types:


1.Closed Range Operator: 
The closed range operator (a...b) is used to get a range that runs from a to b, and includes the values a and b. The value of a must not be greater than b.


for index in 1...5 {
    print("\(index) ") 
}
//it will print 1 2 3 4 5 in new lines.


2.Half-Open Range
A half-open range includes all the values from the lower bound to the upper bound. However,
It ignores the upper bound
It is declared using the ..< operator. For example,
for numbers in 1..<5 {
  print(numbers)
}
// output will be 1 2 3 4 in new lines(5 ge ignored)
3.One-sided Range
We can create a one-sided range using either of the ... or the ..< operator.
A one-sided range contains elements up to infinite in one direction. For example,
let range1 = ..<2 here range1 contains all elements from - infinity to 2

Ans:

var odd = [Int]()


for i in items {
    switch i % 2 { 
    case 0: 
        even.append(i)
    case 1, -1: 
        odd.append(i)
    default: 
        break
    } }

7.Given an array of strings, create a dictionary by grouping them based on 1st letter
for example: [Ranbir, Ranveer, Akshay, Varun, Abhishek, Navneet]
Dictionary = ["R": ["Ranbir", "Ranveer"], "A":["Akshay", "Abhishek"], "N":["Navneet"], "V":["Varun"]]
Also, print the count of strings for each key of the dictionary.




let names = ["Ranbir", "Ranveer", "Akshay", "Varun", "Abhishek", "Navneet"]
var groupedDictionary = [String: [String]]()


for name in names {
    let firstLetter = String(name.prefix(1))
    
    groupedDictionary[firstLetter, default: []].append(name)
}


print(groupedDictionary)


for (key, list) in groupedDictionary {
    print("\(key): \(list.count)")
}
