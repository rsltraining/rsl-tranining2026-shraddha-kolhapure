Q1.Read about static and class keywords used for defining functions in Swift. Explain the difference between them.




Static function cannot be overridden by subclass but class function can be overridden .
Class function is mostly used when we want to perform inheritance and want child class can change behaviour of parent class method


Eg:


 Class Employee{
class func companyPolicy()
{
print(“Follow office time”)
} }
Class Developer:Employee
{
Override class func companyPolicy()
{
print(“developer can work flexible hours) 
} }  


Here the developer class overriding the parent class but if we use static function it will show error as static function cannot be overridden.

        case Burger(Int)

Q4.Create a structure named “Stack” it has a variable (array of integers).
a. Create a function “push” which will take an integer as an input . This function will add the input in the beginning of the array
b. Create a function “pop” which will return and delete the first element of the array and will return nil in case no element in the array are present Try this and see what happens. In case of any issue find out why the issue is occurring and how you can resolve the issue.




struct Stack{
var a:[Int]=[]
func pushitem( _ input : Int ){
        a.insert(input , at : 0)
}


func PrintArray()
{
    for i in a{print(i)}
}


func popitem(){
     if a.count == 0 {return }
    else
    {
    a.remove(at : 0)
    }} }




The problem is here function is written inside struct so struct will going to change which causing error so solution for this is using mutating keyword by marking function as mutating here is correct solution


struct Stack{
var a:[Int]=[]
mutating func pushitem( _ input : Int ){
        a.insert(input , at : 0)
}


func PrintArray()
{
    for i in a{print(i)}
}
mutating func popitem(){
        if a.count == 0 {return }
    else
    {
    a.remove(at : 0)
    }} }


var s=Stack()
s.pushitem(10)
s.pushitem(15)
s.pushitem(20)
s.popitem()
s.PrintArray() //output will get 10 15
